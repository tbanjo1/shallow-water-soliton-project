1. Objective

The goal of this project is to simulate nonlinear wave equations using numerical methods. The project focuses on the Korteweg–de Vries equation and the shallow water equations. For each problem, the spatial derivatives are discretized first, which changes the PDE into a system of ODEs that can be solved in Julia.

ut + 6uux + uxxx = 0

This equation is the KdV equation. It is used in Parts 1 and 2 to study soliton waves.

2. Numerical Method

The method of lines was used. First, the spatial domain was divided into a periodic grid. Then periodic finite-difference matrices were used to approximate the first derivative and the third derivative.

ut = -6uDxu - Dxxxu

For the KdV simulations, the stiff solver Rodas4P() was used because the third derivative term can make the system stiff. For the shallow water equations, the nonstiff adaptive solver Tsit5() was used.

The one-soliton solution used in the project is

s(x,t;c) = c/2 sech2((√c / 2)(x - ct))

3. Part 1: Single-Soliton KdV Simulation

In Part 1, the KdV equation was solved on the periodic domain [-20, 20] using 300 grid points. The initial condition was the single soliton solution with c = 1. The simulation was run from t = 0 to t = 10.

The required output times were:

  • t = 0
  • t = 2
  • t = 4
  • t = 6
  • t = 8
  • t = 10

5. Part 2: Two-Soliton Interaction

In Part 2, the KdV equation was solved again on [-20, 20] using 300 grid points. The initial condition was the sum of two solitons:

u(x,0) = s(x,0;1) + s(x + 8,0;2)

The second soliton starts near x = -8 and has speed parameter c = 2. Since it is faster than the soliton with c = 1, it eventually overtakes the slower soliton. The solution was saved at 81 equally spaced times in [0, 10].

Figure 3. Heatmap of the two-soliton interaction over space and time.

6. Part 2 Discussion

The heatmap shows the faster soliton starting on the left, catching the slower soliton, and passing through it. After the interaction, both waves continue moving while mostly keeping their soliton shapes. This matches the expected behavior of soliton waves.

7. Part 3: Shallow Water Equations

In Part 3, the shallow water equations were solved on the periodic domain [-8, 8] using 400 grid points. The equations are

ht + ((h - δ)u)x = 0
ut + uux = -hx - εu

The bottom depth function was

δ(x) = -2 + 1.75e-2(x - 4)2

The initial conditions were

h(x,0) = e-x2/2,    u(x,0) = 0

The friction parameter was ε = 0.7. The variables h and u were packed into one vector v = [h; u] so that the ODE solver could solve the system.

Figure 4. Animated GIF showing h versus x on top and u versus x on bottom.

8. Part 3 Discussion

The animation shows the wave spreading outward from the center of the domain. As the wave reaches the bump near x = 4, it interacts with the bottom surface and separates into smaller waves. As time increases, the friction term causes the wave motion to settle.

9. Conclusion

This project shows how finite-difference methods and ODE solvers can be used to study nonlinear wave behavior. The KdV simulations showed soliton waves traveling while mostly keeping their shape. The two-soliton simulation showed the faster soliton overtaking the slower one. The shallow water simulation showed a wave interacting with a bottom bump and slowing down because of friction.

Overall, the method of lines gave a clear way to turn each PDE into a system of ODEs that could be solved numerically in Julia.
