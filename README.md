## 📌 Overview
This project simulates soliton-like wave behavior using the Shallow Water Equations (SWE). The model uses numerical methods to study wave propagation, interaction, and stability over time.

---

## 🧠 Mathematical Model
The Shallow Water Equations describe fluid motion under gravity:

- Conservation of mass
- Conservation of momentum

These equations are solved numerically using finite difference methods with periodic boundary conditions.

---

## ⚙️ Methods Used
- Finite Difference Method (FDM)
- Explicit time stepping schemes
- Periodic boundary conditions
- CFL condition for stability
- Gaussian initial conditions for wave generation

---

## 🧪 Features
- 1D wave propagation simulation
- Soliton-like behavior visualization
- Adjustable parameters (grid size, time step, wave height)
- Plotting and animation of wave evolution
